### Installation

1. Clone the repository:
   `git clone https://github.com/akabaytar/unsplash-images.git`
2. Navigate to the project directory:
   `cd unsplash-images`
3. Install dependencies:
   `npm install`
4. Run the project locally:
   `npm run dev`
5. Visit localhost:
   `http://localhost:5173/`
